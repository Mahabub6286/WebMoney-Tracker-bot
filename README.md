# WebMoney Telegram Bot

This is a Telegram bot to track IPweb/WebMoney earnings.

## Features
- Auto detect payments from messages
- Balance tracking
- History tracking
- Admin controls

## Commands
- /report
- /history
- /list
- /add
- /less
- /reset

## Deploy
Use Render or Railway

## Environment Variables
- TELEGRAM_BOT_TOKEN
- ADMIN_TELEGRAM_ID
